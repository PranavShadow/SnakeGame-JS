# 🐍 Snake Game

A classic Snake Game built using **HTML, CSS, and JavaScript**. Control the snake, collect food, increase your score, and try to beat your highest score!

## Preview

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript

## Controls

| Key   | Action     |
| ----- | ---------- |
| ↑ / W | Move Up    |
| ↓ / S | Move Down  |
| ← / A | Move Left  |
| → / D | Move Right |

## Gameplay

Guide the snake around the board, eat food to gain points, and avoid hitting the walls. The game ends when the snake collides with a boundary.
